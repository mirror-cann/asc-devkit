# 入门样例介绍

## 概述

基于Ascend C的简单SIMT算子实现示例，展示SIMT编程基本语法、编译及基于<<<>>>的调用方法。

## 样例列表

|  目录名称                                                   |  功能描述                                              |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| [00_quickstart](./00_quickstart) | 基于SIMT编程方式实现HelloWorld样例，是SIMT编程的核函数、`<<<>>>`调用等基础代码示例 |
| [01_gather](./01_gather) | 基于SIMT编程方式实现gather样例，展示了从简化场景到泛化场景的离散访存类算子开发方法 |
