# Introduction Example Overview

## Overview

Simple SIMT operator implementation examples based on Ascend C, demonstrating SIMT programming basic syntax, compilation, and <<<>>> based invocation methods.

## Example List

|  Directory Name                                                   |  Description                                              |
| ------------------------------------------------------------ | ---------------------------------------------------- |
| [00_quickstart](./00_quickstart) | HelloWorld example implemented based on SIMT programming, basic code example for SIMT programming kernel functions, <<<>>> invocation, and so on |
| [01_gather](./01_gather) | Gather example implemented based on SIMT programming, demonstrating the development method for discrete memory access operators from simplified scenarios to generalized scenarios |
