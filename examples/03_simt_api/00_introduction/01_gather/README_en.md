# Programming Basics Sample Introduction

## Overview

This document presents basic samples based on Ascend C SIMT programming, demonstrating the SIMT programming operator development user interface, development process, and more.

## Sample List

| Directory Name | Description |
| --------- | --------- |
| [basic_gather](./basic_gather) | This sample implements a Gather operator using Ascend C SIMT programming, demonstrating the development of operators with discrete memory access patterns in simplified scenarios. |
| [general_gather](./general_gather) | This sample implements a Gather operator for generalized scenarios using Ascend C SIMT programming, demonstrating the end-to-end process of SIMT operator development. |
