# Gather样例介绍

## 概述

基于Ascend C SIMT编程的基础样例，包含简化场景和泛化场景的Gather算子，展示基于SIMT编程的算子开发接口、开发流程等。

## 样例列表

| 目录名称  | 功能描述 |
| --------- | --------- |
| [basic_gather](./basic_gather) | 样例基于Ascend C SIMT编程方式实现简化场景的Gather算子，展示SIMT编程的基本流程。 |
| [general_gather](./general_gather) | 样例基于Ascend C SIMT编程方式实现泛化场景的Gather算子，展示SIMT算子开发的端到端流程。 |
