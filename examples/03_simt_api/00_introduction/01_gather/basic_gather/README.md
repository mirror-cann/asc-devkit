# Basic Gather样例介绍

## 概述

基于Ascend C SIMT编程的基础样例，包含简单场景（固定shape）的一维Gather算子和二维Gather算子，展示SIMT编程的基本流程。

## 样例列表

| 目录名称  | 功能描述 |
| --------- | --------- |
| [gather_1d](./gather_1d) | 样例基于Ascend C SIMT编程方式实现固定shape的一维Gather算子。 |
| [gather_2d](./gather_2d) | 样例基于Ascend C SIMT编程方式实现固定shape的二维Gather算子。 |
