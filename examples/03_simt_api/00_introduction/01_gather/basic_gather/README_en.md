# Basic Gather Sample Introduction

## Overview

Basic samples based on Ascend C SIMT programming, implementing one-dimensional and two-dimensional Gather operators for simple scenarios with fixed shapes, demonstrating the basic SIMT programming flow.

## Sample List

| Directory Name | Description |
| --------- | --------- |
| [gather_1d](./gather_1d) | This sample implements a fixed-shape one-dimensional Gather operator using Ascend C SIMT programming. |
| [gather_2d](./gather_2d) | This sample implements a fixed-shape two-dimensional Gather operator using Ascend C SIMT programming. |
