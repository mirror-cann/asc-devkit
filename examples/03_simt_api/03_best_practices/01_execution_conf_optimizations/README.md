# Execution Configuration Optimizations样例介绍

## 概述

算子执行配置优化，通过<<<>>>直调的实现方式，介绍SIMT编程方式下的gridDim/blockDim相关配置。当前提供了设置最大线程块的优化案例，用于充分利用寄存器，优化算子执行性能。

## 样例列表

| 目录名称                                               | 功能描述                                                                                        |
| ------------------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| [sincos_compute](./max_thread_config/sincos_compute)   | 以sincos的计算为例，对比\_\_launch_bounds__使用默认值和512两种场景下的性能差异，演示SIMT编程方式下寄存器溢出对于性能的影响以及最大线程块的配置方法。 |
