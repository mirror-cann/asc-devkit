# Execution Configuration Optimizations样例介绍

## 概述

算子执行配置优化，通过<<<>>>直调的实现方式，介绍SIMT编程方式下的gridDim/blockDim相关配置。当前提供了设置最大线程数的优化案例（sincos_compute），用于充分利用寄存器，优化算子执行性能；以及线程块数量配置策略样例（grid_dim_config），展示不同数据量场景下的线程块配置优化方法。

## 样例列表

| 目录名称                                               | 功能描述                                                                                        |
| ------------------------------------------------------ | ----------------------------------------------------------------------------------------------- |
| [sincos_compute](./max_thread_config/sincos_compute)   | 以sincos的计算为例，对比\_\_launch_bounds__使用默认值和512两种场景下的性能差异，演示SIMT编程方式下寄存器溢出对于性能的影响以及最大线程数的配置方法。 |
| [grid_dim_config](./grid_dim_config)   | 样例以Gather算子为例，在不同数据量场景下，展示不同的线程块配置策略对算子性能的影响，并提供相应的优化指导。 |
