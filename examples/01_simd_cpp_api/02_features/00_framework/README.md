# Framework样例介绍

## 概述

本样例介绍了PyTorch、TensorFlow、ONNX框架、GE原生构图以及ACLGraph capture场景的自定义算子实现方法。

## 样例列表

| 目录名称  | 功能描述 |
| --------- | --------- |
| [00_pytorch](./00_pytorch) | 样例介绍PyTorch框架的自定义算子实现方法 |
| [01_tensorflow](./01_tensorflow) | 样例介绍TensorFlow框架的自定义算子实现方法 |
| [02_onnx](./02_onnx) | 样例介绍ONNX模型调用相关的自定义算子实现方法 |
| [03_ge](./03_ge) | 样例介绍Ascend C算子接入GE实现方法 |
| [04_aclgraph](./04_aclgraph) | 样例介绍通过ACL Model RI接口捕获并重复执行Ascend C运行时任务的方法 |
