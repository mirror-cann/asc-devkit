# Compilation Sample Introduction

## Overview

This sample introduces the implementation methods for custom operator compilation, packaging, and deployment based on Ascend C.

## Sample List

| Directory Name | Description |
| -------------- | ----------- |
| [custom_op](./custom_op) | This sample introduces the workflow of custom operator compilation, packaging, and deployment |
| [custom_op_static_lib](./custom_op_static_lib) | This sample introduces the compilation and invocation method for custom operator static libraries |
| [parallel_ops_package](./parallel_ops_package) | This sample introduces the implementation of parallel compilation, packaging, and deployment for multiple vendors |
