## Windows平台压缩工具推荐

### 方案一：PNGGauntlet（推荐，无损压缩）

**优点**：
- ✅ 无损压缩，不降低图片质量
- ✅ 压缩效果显著（30-50%）
- ✅ 批量处理
- ✅ 免费开源

**下载地址**：
https://pnggauntlet.com/

**使用步骤**：
1. 下载安装 PNGGauntlet
2. 添加上述6个图片文件
3. 点击"Compress"开始压缩
4. 压缩完成后替换原文件

---

### 方案二：TinyPNG（在线工具）

**优点**：
- ✅ 有损压缩，压缩效果显著（50-70%）
- ✅ 网页操作，无需安装软件
- ✅ 批量处理（最多20张）
- ✅ 保持透明度

**网址**：
https://tinypng.com/

**使用步骤**：
1. 打开 https://tinypng.com/
2. 上传6个图片文件（可拖拽上传）
3. 等待压缩完成
4. 点击"Download all"下载压缩后的图片
5. 替换原文件

---

### 方案三：ImageMagick（命令行工具）

**优点**：
- ✅ 强大的命令行工具
- ✅ 支持批量压缩
- ✅ 跨平台

**下载地址**：
https://imagemagick.org/script/download.php#windows

**压缩命令**（PowerShell）：

```powershell
# 进入图片目录
cd C:\work_for_code\doc_design\asc-devkit\docs\guide\figures

# 批量压缩PNG图片（无损压缩）
$images = @(
    "矩阵计算单元架构图.png",
    "读读冲突优化.png",
    "读读冲突.png",
    "核间同步业务场景示例.png",
    "地址分配优化前示意图.png",
    "地址分配优化后示意图.png"
)

foreach ($img in $images) {
    magick convert $img -quality 85 -strip "compressed_$img"
    Write-Output "Compressed: $img"
}

# 压缩完成后，手动替换原文件
```

## 压缩后验证

### 使用PowerShell验证压缩效果

```powershell
# 进入图片目录
cd C:\work_for_code\doc_design\asc-devkit\docs\guide\figures

# 检查压缩后的图片大小
$images = @(
    "矩阵计算单元架构图.png",
    "读读冲突优化.png",
    "读读冲突.png",
    "核间同步业务场景示例.png",
    "地址分配优化前示意图.png",
    "地址分配优化后示意图.png"
)

foreach ($img in $images) {
    if (Test-Path $img) {
        $file = Get-Item $img
        $sizeKB = [math]::Round($file.Length / 1KB, 2)
        if ($sizeKB < 100) {
            Write-Output "$img: $sizeKB KB (✅ OK)"
        } else {
            Write-Output "$img: $sizeKB KB (⚠️ Still large)"
        }
    }
}
```

## 注意事项

### ⚠️ 压缩前备份

建议在压缩前备份原图：

```powershell

# 备份原图
```

### ⚠️ 质量损失

- **无损压缩**（PNGGauntlet/OptiPNG）：不损失质量，压缩效果30-50%
- **有损压缩**（TinyPNG）：轻微损失质量，压缩效果50-70%

建议：
- 对于架构图等重要图片，优先使用**无损压缩**
- 对于示意图等，可使用**有损压缩**
