# 接口变更说明<a name="ZH-CN_TOPIC_0000002547538511"></a>

为了优化API命名，使其更加清晰和易于理解，当前版本相比于CANN8.5.0及前序版本，部分API命名有变更，具体变更情况请参考以下表格：

| 原接口名 | 新接口名 | 备注 |
|---|---|---|
| ScalarGetCountOfValue | [GetBitCount](../SIMD-API/基础API/标量计算/GetBitCount.md) | 无 |
| ScalarCountLeadingZero | [CountLeadingZero](../SIMD-API/基础API/标量计算/CountLeadingZero.md) | 无 |
| ScalarGetSFFValue | [GetSFFValue](../SIMD-API/基础API/标量计算/GetSFFValue.md) | 无 |
| ScalarCast | [Cast（float转half、int32_t）](../SIMD-API/基础API/标量计算/Cast（float转half-int32_t）.md) | 无 |
| ToBfloat16 | [Cast（float转bfloat16_t）](../SIMD-API/基础API/标量计算/Cast（float转bfloat16_t）.md) | 无 |
| ToFloat | [Cast（多类型转float）](../SIMD-API/基础API/标量计算/Cast（多类型转float）.md) | 其中一个重载版本的返回值类型由float改为模板U, U = float |
| FusedMulAddRelu | [MulAddRelu](../SIMD-API/基础API/Memory矢量计算/复合计算/MulAddRelu.md) | 无 |
| CastDeq | [CastDequant](../SIMD-API/基础API/Memory矢量计算/复合计算/CastDequant.md) | 无 |
| CompareScalar | [Compares](../SIMD-API/基础API/Memory矢量计算/比较与选择/Compares.md) | 无 |
| GetAccVal | [GetReduceRepeatSumSpr](../SIMD-API/基础API/Memory矢量计算/归约计算/寄存器辅助接口/GetReduceRepeatSumSpr(ISASI).md) | 无 |
| GetReduceMaxMinCount | [GetReduceRepeatMaxMinSpr](../SIMD-API/基础API/Memory矢量计算/归约计算/寄存器辅助接口/GetReduceRepeatMaxMinSpr(ISASI).md) | 无 |
| SetAtomicNone | [DisableDmaAtomic](../SIMD-API/基础API/原子操作/DisableDmaAtomic.md) | 无 |
| InitConstValue | [Fill](../SIMD-API/基础API/矩阵计算（ISASI）/矩阵计算的搬入/辅助配置接口/Fill.md) | 无 |
| SetMMLayoutTransform | [SetMMRowMajor](../SIMD-API/基础API/矩阵计算（ISASI）/矩阵计算/SetMMRowMajor.md)/[SetMMColumnMajor](../SIMD-API/基础API/矩阵计算（ISASI）/矩阵计算/SetMMRowMajor.md) | SetMMLayoutTransform(true)对应SetMMRowMajor()。SetMMLayoutTransform(false)对应SetMMColumnMajor()。 |
| SetHF32Mode | [SetHF32Mode](../SIMD-API/基础API/矩阵计算（ISASI）/矩阵计算/SetHF32Mode.md) | 接口名不变，参数类型由bool修改为枚举HF32Mode。 |
| SetHF32TransMode | [SetHF32TransMode](../SIMD-API/基础API/矩阵计算（ISASI）/矩阵计算/SetHF32TransMode.md) | 接口名不变，参数类型由bool修改为枚举HF32TransMode。 |
| CalcTschBlockDim | [CalcTschNumBlocks](../Utils-API/平台信息获取/PlatformAscendC/CalcTschNumBlocks.md) | 无 |

以下表格的接口于9.1.0修改，相比于CANN9.0.0及前序版本，部分API命名有变更，具体变更情况请参考以下表格：

| 原接口名 | 新接口名 | 备注 |
|---|---|---|
| BlockReduceSum / BlockReduceMax / BlockReduceMin | [ReduceDataBlock](../SIMD-API/基础API/Memory矢量计算/归约计算/ReduceDataBlock.md) | &bull;提供统一的DataBlock归约操作接口，通过模板参数区分不同归约操作 <br>&bull;调整mask与repeatTime的参数顺序 |
| WholeReduceSum / WholeReduceMax / WholeReduceMin | [ReduceRepeat](../SIMD-API/基础API/Memory矢量计算/归约计算/ReduceRepeat.md) | 提供统一的Repeat归约操作接口，通过模板参数区分不同归约操作 |
| RepeatReduceSum | [ReduceRepeat](../SIMD-API/基础API/Memory矢量计算/归约计算/ReduceRepeat.md) | &bull;提供统一的Repeat归约操作接口，通过模板参数区分不同归约操作 <br>&bull;调整mask与repeatTime的参数顺序 <br>&bull;删除无效参数dstBlkStride |
| PairReduceSum | [ReducePairElem](../SIMD-API/基础API/Memory矢量计算/归约计算/ReducePairElem.md) | &bull;提供统一的Pair元素归约操作接口，通过模板参数区分不同归约操作 <br>&bull;调整mask与repeatTime的参数顺序 |
