# 概述<a name="ZH-CN_TOPIC_0000002623072453"></a>

**表1** MaskReg计算接口

| API | 描述 |
| --- | --- |
| [Move](Move.md) | 将src中的元素复制到dst中的对应位置。 |
| [Interleave](Interleave.md) | 将src0和src1中的元素交织存入dst0和dst1中。 |
| [DeInterleave](DeInterleave.md) | 将src0和src1中的元素解交织存入dst0和dst1中。 |
| [Select](Select.md) | 从src0和src1中，根据mask的比特位值选取元素，得到目的操作数dst。 |
| [Pack](Pack.md) | 将源操作数src的偶数位bit，提取到目的操作数dst的低半部分或高半部分。 |
| [UnPack](UnPack.md) | 将源操作数src的低半部分或高半部分，展开到目的操作数dst。 |
| [MoveMask](MoveMask.md) | 从[SetVectorMask](../../Memory矢量计算/掩码操作/SetVectorMask.md)设置的掩码寄存器{MASK1, MASK0}中读取Mask值，并按模板参数T对应的数据格式转换后写入返回值MaskReg。 |
