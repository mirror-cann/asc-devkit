## 功能说明

在Conv3D计算完成后，必须调用一次End函数，以清除EventID并释放内部申请的临时内存。

## 函数原型

```
void End()
```

## 参数说明

无

## 返回值说明

无

## 约束说明

End接口必须在IterateAll接口后调用。

## 调用示例

```
conv3dApi.End();
```
